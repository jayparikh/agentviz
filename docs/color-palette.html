Color Family Summary

All cool-toned. Zero browns or oranges.

Blue
explore, tool_call
Green
task, output, success
Purple
general, context, system
Cyan
code-review, agents
Pink
configure-copilot
Slate
reasoning, user
Yellow
warning only
Rose
error only
← Reserved for semantic states, never used as track/agent identity colors

Design rules:
• 6 identity hues: blue, green, purple, cyan, pink, slate -- used for agents, tracks, and data
• 2 semantic hues: yellow (warning), rose (error) -- never used as identity colors
• Accent: single blue (#6475e8) shared across both modes
• Light variants are deeper/more saturated for contrast on white
• No browns. No oranges. All cool-toned.
